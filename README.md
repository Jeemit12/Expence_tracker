# 💰 Expense Tracker using Python (Tkinter + SQLite)

A modern and interactive **Expense Tracker Application** built using **Python**, **Tkinter GUI**, and **SQLite database**.  
This application helps users manage daily expenses, track spending, and visualize financial data efficiently.

## 🎮 Application Overview

- Secure login & registration system 🔐  
- Add, edit, and delete expenses 💸  
- Categorize expenses (Food, Travel, Shopping, etc.) 🧾  
- Monthly budget tracking 💼  
- Visual analytics with pie charts 📊  
- Export data as CSV or TXT 📁  
- Dashboard with real-time insights 📈
